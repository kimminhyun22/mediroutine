# 메디루틴

약 먹는 시간을 잊지 않도록 복약 루틴, OCR 입력, 가족 안심 공유 흐름을 보여주는 웹서비스 프로토타입입니다.

## 실행

컴퓨터에서 확인할 때는 `메디루틴 실행.command`를 더블클릭한 뒤 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:4173
```

## 휴대폰 설치

GitHub Pages로 배포한 뒤 휴대폰에서 배포 주소를 열면 홈 화면에 추가할 수 있습니다.

- 아이폰: Safari 공유 버튼 → 홈 화면에 추가
- 안드로이드: Chrome 메뉴 → 앱 설치 또는 홈 화면에 추가
